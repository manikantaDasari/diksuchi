How it works

One proxy. Every model.
Zero code changes.

Point your existing OpenAI SDK at Diksuchi. It handles the routing — no rewrites, no vendor lock-in, no surprises.

01

Point your SDK at Diksuchi

Change one line — base_url — to localhost:8080. Works with any OpenAI-compatible SDK, tool, or framework.

02

Diksuchi reads the prompt

Analyses your request against 13 rules — complexity keywords, token count, regex patterns, time of day — in under 1ms. Fully local.

03

Routed to the best model

Simple questions stay local — free, private, instant. Complex tasks go to GPT-4o, Claude, or Gemini. Automatically, every time.

Features

Everything you need.
Nothing you don't.

OpenAI-compatible

Drop-in replacement. Works with any SDK or tool that speaks the OpenAI API — LangChain, Continue.dev, Cursor, your own code.

Local-first by default

Routes to your local Ollama models whenever possible. Save up to 90% on API costs without changing how you work.

Zero code changes

Change one URL. That's it. No SDK wrappers, no middleware to maintain, no provider-specific logic scattered in your app.

10+ providers

OpenAI, Anthropic, Groq, Gemini, Together, Mistral, Ollama, LM Studio — all configured from a single YAML file.

Privacy by design

Sensitive prompts stay local. Credential patterns are detected and never forwarded to a third-party provider.

Live dashboard

Real-time routing decisions, request log, provider switcher, and cost visibility — built in, zero setup required.

Quick start

Up in 60 seconds.

Docker is the fastest path. No Python, no pip, no environment setup.

1

Clone and configure

Copy .env.example → .env and add your OPENAI_API_KEY. All other providers are optional.

2

Start the full stack

Pulls Ollama, downloads llama3.2, starts the router. One command — everything wires up automatically.

3

Point your SDK at Diksuchi

Set base_url to http://localhost:8080/v1. Use api_key="diksuchi" — any string works.

4

Open the dashboard

Visit localhost:8080 to watch every routing decision happen in real time.
